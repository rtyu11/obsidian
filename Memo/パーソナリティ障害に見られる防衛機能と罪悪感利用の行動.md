# パーソナリティ障害に見られる防衛機能と罪悪感利用の行動

## 1. 防衛機能の基本

防衛機制は、不安や葛藤に直面したときに心を守るための自動反応。
健康な人なら「落ち込むけど反省して修正する」という形で機能する。
パーソナリティ障害では「自己像を守るために」極端な形で発動し、周囲からは「理解不能」な行動として映る。

## 2. 主な防衛機能

### 否認（Denial）
不都合な現実を「存在しない」として処理する。→ 一時的に自尊心を守るが改善が進まない。

### 投影（Projection）
自分が抱えている攻撃性や劣等感を「相手がそう思っている」と変換する。→ 関係性を悪化させやすい。

### 分裂（Splitting）
他人を「完全に良い人／完全に悪い人」と二分する。些細な不満で全否定に転じる。

### 合理化（Rationalization）
自分の行動に後付けの理由をつけて正当化する。→ 実態からどんどん離れていく。

### 被害者化（Victim stance）
自分を弱く演出し、相手に「責めすぎたかも」と罪悪感を抱かせる。→ 本人には「攻撃している」という自覚がほとんどない。

### 受動攻撃（Passive-aggression）
表向きは従順に見せつつ、遅刻・無視・曖昧返答などで相手を困らせる。→ 直接対立の恐怖を避けながら攻撃する。

## 3. 罪悪感を利用する心理

罪悪感は人間関係で非常に強い「行動修正のトリガー」。パーソナリティ障害の人は「相手に罪悪感を抱かせる」ことで状況を変える。

**心理の流れ：**
1. 注意や批判を受ける → 強烈な不安と恥の感情が生じる
2. 直視できない → 「自分は被害者だ」という物語に切り替える
3. 落ち込む姿や自己卑下を演出 → 相手が「責めすぎたか」と罪悪感を抱く
4. 相手の譲歩や優しさを引き出す → 自分の立場を回復

## 4. 全体像

**「本人は自分を守っているつもりでも、結果として他人を攻撃している」という二重性が理解の鍵。**
被害者ポジションは「防御と攻撃のハイブリッド」。表面上は弱く見えても、裏では罪悪感を利用した強力な操作になっている。
