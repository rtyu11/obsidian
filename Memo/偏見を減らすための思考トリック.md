# 偏見を減らすための思考トリック（個人化）

## 1. ステレオタイプ化（Stereotyping）
人は他人を素早く理解するために、上司・外国人・高齢者などの**カテゴリー（ラベル）**で処理する。
→ これが**無意識の偏見（Implicit Bias）**の原因になる。

## 2. カテゴリー処理（Category-based processing）
脳が「その人個人」ではなく**グループの典型像**で判断する認知処理。
例：「上司っぽい人」「クレーマーっぽい人」

## 3. 個人化（Individuation）
相手をカテゴリーではなく**一人の個人として認識する認知モード**。
個人の好み・性格・過去などを想像すると発動する。偏見を減らす研究でよく使われる方法。

## 4. 心的シミュレーション（Mental Simulation）
相手の生活や性格を**頭の中で具体的に想像すること**。
- この人はどんな野菜が嫌いだろう
- どんなポケモンを持っていそうか

→ **抽象的なラベル → 具体的な人物像**に認知が切り替わる。

## 5. 再人間化（Re-humanization）
相手を「役割・属性」ではなく**生活や感情を持つ人間として認識する状態**。
→ 共感（Empathy）が増え、偏見が減る。

## 思考トリックの構造

1. 人は他人を**カテゴリー化**する
2. カテゴリー化は**偏見**を生む
3. 相手の個人的特徴を**想像**する
4. 脳が**個人化モード**に切り替わる
5. 偏見が弱まる

## 「野菜」や「ポケモン」を使う理由
- 誰にでもある**個人的な好み**を想像させる
- **具体的な人物像**を作る
- 相手の**性格・生活・経験**を推測する
→ 脳の処理が**ラベル認知 → 人物認知**に変わる。
