# 運送業界の採用困難度とコスト構造に関する分析資料
## 1. 採用難易度：全産業平均の「2.2倍」という異常値
ドライバー確保の難しさは、もはや従来の求人媒体だけで解決できる水準を超えています。
 * **有効求人倍率（2024年12月時点）** [1]
   * **自動車運転職：2.82倍**
   * 全職業平均：1.27倍
   * ⇒ 他業種と比較して「2.2倍」人が集まりにくい構造です。
 * **採用実感（2024年調査）**
   * 採用が「厳しかった」と回答した企業の割合：**61.1%**
   * 全業種の中でワースト1位。2024年問題の影響で前年より18ポイントも急増しています。
## 2. 採用コストの「インフレ」：他業界を圧倒する投資額
「人が集まらない」ために、運送業者は他業種の数倍のコストを投じざるを得ない状況です。
 * **年間採用費用の実績平均（1社あたり）** [2]
   * **運輸・物流・倉庫業：1,459.6万円**
   * 流通・小売・フードサービス業：303.8万円
   * ⇒ 運送業界は、小売業などの**約4.8倍**もの採用コストを支払っています。
 * **1人あたりの獲得単価（正社員）** [3][4]
   * 求人広告利用：**約30.5万円〜35万円**
   * 人材紹介（エージェント）：**100万円〜150万円**
   * ※成功報酬（年収の30-35%）への依存が、1人あたり100万円超えのコストを常態化させています。
## 3. 「2024年問題」がもたらした構造的危機
人手不足は単なる「欠員」ではなく、事業継続そのものを揺るがすリスクです。
 * **輸送能力の不足予測**: 2030年には**34%の荷物が運べなくなる**と試算されています [5]。
 * **賃金上昇の圧力**: 2024年度のドライバー1カ月平均賃金は前年比**7.4%増**。採用競争に勝つための「提示条件」自体も年々上がっています 。
 * **価格転嫁の難しさ**: 物流コスト増を運賃に転嫁できている企業は**32.3%**に留まり、原材料費（56.6%）に比べ大幅に遅れています 。
## 4. 解決策：獲得単価を1/10に抑える「新募集アプローチ」のROI
既存の「高額広告・紹介」から、低コスト・高定着な手法へシフトする必要があります。
| 採用手法 | 期待される獲得単価 | 特徴・メリット [6][7] |
|---|---|---|
| **人材紹介（既存）** | **100〜150万円** | 確実性は高いが、1人採用するごとに経営利益が大幅に削られる。 |
| **リファラル採用** | **約15万円** | 紹介手数料＋活動費。**コストを約85〜90%削減**。ミスマッチが少なく定着率も高い。 |
| **SNS採用運用** | **月20〜30万円** | TikTokやインスタを活用。潜在層へリーチし、中長期的に自社資産として応募を自動化。 |
| **特化型サイト** | **7万円台〜** | ドライバー専門媒体（ドラEVER等）の活用。大手サイトより高いマッチング率 。 |
### 提言：新規アプローチの正当性
現在の運送業界は「全業種で最も採用が難しく、かつ最もコストがかかる」市場です。
1人150万円を投じる人材紹介に依存し続けることは、営業利益率の低い運送業において致命的な経営リスクです。**「獲得単価を15万円〜20万円台へ抑える独自の募集ルート（SNS・リファラル）」**を構築することこそが、2024年問題以降の勝ち残り戦略における最優先事項です。
