# Vault Updater Skill

## トリガー条件
- **「更新分をまとめて」**：新規・更新ハイライトを検出し、タグ・リンクを自動付与してBooksノートを作成・更新する
- **「〇〇について話したい」**：指定した本のBooksノートをベースに会話モードに入り、気づきをノートに追記する
- **「Insightにまとめて」**：会話から得た気づきを `Insights/` フォルダにまとめる

---

## フォルダ構成

```
Source/Kindle/    ← Kindleハイライト原文（編集禁止）
Source/OCR/       ← OCRハイライト原文（編集禁止）

Notes/Books/      ← 本1冊ごとのノート
                     ハイライト原文＋会話からの気づきをまとめたもの

Notes/Topics/     ← キーワード・ジャンルごとの横断ノート
                     複数の本にまたがる気づきのまとめ

Insights/         ← 会話から生まれた気づきのまとめ（日付付き）
```

---

## 【トリガー1】「更新分をまとめて」

### 1. 処理対象ファイルの検出
- `/Source/Kindle` と `/Source/OCR` 内のファイルを一覧取得する
- `/Notes/Books` 内の既存ノートを一覧取得する
- 以下を処理対象とする：
  - **新規**：Booksノートがまだ存在しない本
  - **追記あり**：ハイライト数（`highlightsCount`）がBooksノートの記録より増えている本
- 該当なしの場合は「新しい更新はありませんでした」と伝えて終了する

### 2. タグの自動付与

#### ジャンルタグ（NDC日本十進分類法・2階層）
本の内容からジャンルを判断し、大分類と小分類の両方を付与する。

| 大分類タグ | 小分類タグの例 |
|---|---|
| `#genre/哲学` | `#genre/哲学/心理学` `#genre/哲学/倫理学` `#genre/哲学/思想` |
| `#genre/歴史` | `#genre/歴史/日本史` `#genre/歴史/世界史` `#genre/歴史/伝記` |
| `#genre/社会科学` | `#genre/社会科学/経営` `#genre/社会科学/経済` `#genre/社会科学/教育` `#genre/社会科学/法律` |
| `#genre/自然科学` | `#genre/自然科学/医学` `#genre/自然科学/脳科学` `#genre/自然科学/生物学` |
| `#genre/技術` | `#genre/技術/情報工学` `#genre/技術/建築` |
| `#genre/文学` | `#genre/文学/日本文学` `#genre/文学/海外文学` `#genre/文学/エッセイ` |
| `#genre/芸術` | `#genre/芸術/デザイン` `#genre/芸術/音楽` |

#### キーワードタグ（内容から自動抽出）
ハイライト内容を読み取り、主要テーマ・概念を3〜6個タグとして付与する。
例：`#習慣` `#意思決定` `#睡眠` `#リーダーシップ` `#マインドセット` など

### 3. Booksノートの作成・更新

#### 保存先
`/Notes/Books/[本のタイトル].md`

#### ノートフォーマット（新規作成）
```markdown
# [本のタイトル]

著者: [[著者名]]
#genre/大分類/小分類 #キーワード1 #キーワード2 #キーワード3

---

## ハイライト要約

[ソースファイルの全ハイライトを読み込み、主要テーマ・重要な主張を簡潔にまとめた要約（箇条書き可）]

---

## 気づき

<!-- 「〇〇について話したい」で会話後に追記される -->

---

## 関連ノート

- [[関連するBooksノート]]
- [[関連するTopicsノート]]

<!-- highlightsCount: [件数] -->
```

**重要**：
- `## ハイライト要約` はソースファイル（`/Source/Kindle` または `/Source/OCR`）の**全ハイライトを読み込んだうえで**、内容を簡潔にまとめる
- AIによる分析・考察は書かない。あくまでハイライト内容の要約にとどめる
- 「気づき」セクションは空欄のまま残す

#### 追記ありの場合
- ソースファイルの全ハイライトを再読み込みし、`## ハイライト要約` を最新内容で更新する
- `highlightsCount` を最新の件数に更新する
- 「気づき」セクションは編集しない

### 4. 関連リンクの自動付与
- 同じ著者の他のBooksノートがあれば `## 関連ノート` に追記する
- 同じキーワードタグを持つBooksノートがあれば `## 関連ノート` に追記する
- 共通テーマのTopicsノートがあれば `## 関連ノート` に追記する

### 5. Topicsノートの確認・更新
- 今回付与したキーワード・ジャンルタグを持つ本が**2冊以上**存在する場合：
  - 該当するTopicsノートが存在しなければ `/Notes/Topics/[テーマ名].md` を新規作成する
  - 存在する場合は新しいBooksノートへのリンクを追記するだけ（内容の書き換えは禁止）

#### Topicsノートフォーマット（新規作成時）
```markdown
# [テーマ名]

#キーワード #関連キーワード #genre/大分類/小分類

---

## 関連する本

- [[Booksノート名]]
- [[Booksノート名]]

---

## このテーマについての気づき

<!-- 「〇〇について話したい」で会話後に追記される -->
```

### 6. 完了報告
- 作成・更新したBooksノート一覧とタグ
- 作成・更新したTopicsノート一覧
- 「〇〇について話したい」で会話を始められる旨の案内

## 【トリガー2】「〇〇について話したい」

### 1. 対象ノートの読み込み

#### 本について話す場合（「〇〇という本について話したい」）
- 指定された本の `/Notes/Books/[本のタイトル].md` を読み込む（過去の気づきを把握する）
- `/Source/Kindle` または `/Source/OCR` 内の**その本に対応するソースファイル1つ**を読み込む（全ハイライトを把握する）
- Booksノートの `## 関連ノート` にTopicsノートへのリンクがある場合は、そのTopicsノートも読み込む
- Topicsノートに `<!-- 未解決の問い：〜 -->` が含まれている場合は、**会話の冒頭でその課題を提示する**

#### テーマ・タグについて話す場合（「〇〇というテーマについて話したい」）
- 対応する `/Notes/Topics/[テーマ名].md` を読み込む
- Topicsノートに紐づく各Booksノートの**ソースファイル**（`/Source/Kindle` または `/Source/OCR`）を読み込む

### 2. 会話の進め方
- ハイライト内容を把握したうえで会話を開始する
- AIは自分の意見・解釈を積極的に述べてよい
- 会話の末尾に毎回質問を投げかける必要はない。自然な会話の流れに任せる
- 会話は自然なペースで続ける。まとめを急かさない

### 3. 会話終了後の追記
ユーザーが「まとめて」「ノートに残して」などと言ったら：

**追記ルール（重要）**：
1. **ユーザーが自分で語った言葉**はそのまま含める
2. **AIが提案・整理した内容**はユーザーが明示的に同意・肯定した場合のみ含める（同意なく含めない）
3. AIが補足・解釈・言い換えをして水増ししない

追記先は **本について話した場合** → Booksノートの `## 気づき` セクション（日付付き）
追記先は **テーマについて話した場合** → 対応するTopicsノートの `## このテーマについての気づき` セクション（日付付き）

#### 追記フォーマット
```markdown
### [YYYY-MM-DD]
[ユーザーの言葉、および同意したAIの発言]
```

## 【トリガー3】「Insightにまとめて」

会話の終わりにユーザーが「Insightにまとめて」と言ったら：

### 保存先
`Insights/[YYYY-MM-DD]-[テーマ].md`

### フォーマット
```markdown
# [テーマ・課題名]

日付: [YYYY-MM-DD]
関連: [[参照したBooksノート]] [[参照したTopicsノート]] [[関連するTasks]]

---

## 課題・問い
[会話の起点となった課題やアイデア]

---

## 会話から得た気づき
[ユーザーが自分で語った言葉・AIに同意した内容のみ]

---

## 次のアクション
- [ ] [会話から生まれたやること]
```

**追記ルール：**
- ユーザーが自分で語った言葉はそのまま含める
- AIが提案した内容はユーザーが同意した場合のみ含める

---

## 禁止事項（Important Rules）
- `/Source` 配下のファイルは**編集・移動・削除すべて禁止**
- AIはハイライトの要約・考察・分析を**書かない**
- ユーザーが同意していない内容をノートに書かない
- 上記フォルダ以外の新規フォルダ作成禁止
- すべての返答・ノートは**日本語**で行う
